# 🚀 Quickstart 

Run the website locally

```
git clone https://github.com/harshhhdev/geethoob.git
```

## Setting up the project

```bash
cd geethoob

# Install deps
yarn
```

## Starting server

```bash
yarn dev
```

Server should now be running on [localhost](https://localhost:3000)
